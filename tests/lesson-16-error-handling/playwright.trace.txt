
- normal playwright execution and error:
npx playwright test

- playwright execution + traceviewer capture of debug info
npx playwright test --trace on

- or set trace option in playwright.config.test

- traceviewer doc: https://playwright.dev/docs/trace-viewer-intro

- can open traceviewer with command provided after test failure like so:
npx playwright show-trace test-results\scenario.example-Basic-Scenario-Microsoft-Edge\trace.zip

- or through report link
